### 三：如何完成一个简单定投策略

1：无任何限制简单定投，比如：每6个小时定投

```php
//php自带函数 默认秒单位
sleep(60*60*6);
```

2：某一时刻定投，比如每天12:30点准时定投，相隔24小时

```php
//php自带时间函数
$time=time();

//php自带格式化函数
echo date('Y-m-d H:i:s',$time);
```
